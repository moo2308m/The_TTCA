Introduction $COWWD(n)$ is a user-made computable function, made by me, mainly because I like graph minors a lot and I wanted to see how far I could take graph minors before they just become useless. It's a fast growing function and it uses a lot of graph minors, iterated graph minors, and is also bounded by the Extended Rathjen ordinal system ($\Psi_0(\chi_K(M))$). A graph minor is a finite sequence of graphs that's constrained by vertices or edges, based off of the Robertson-Seymour graph minor theorem, and an iterated graph minor is a version of a graph minor that's iterated $var$ times ($IGM^{var}(n)$). It's inspired off of $SCG(n)$, so of course it grows fast. The equation Here's the equation: $$Z = n [\uparrow^{2^{10^{28^n}}}] n + 2^{10^{28}}$$ $$GM(n) = |GraphMinor(n)|$$ $$E^1(x) = GM(x [\uparrow^Z] x)$$ For any value of k > 1: $E^k(x) = GM(E^{(k^x) - 1}(Z) [\uparrow^{k + 1}] E^Z(k - 1))$ $$M = E^Z(GM(Z [\uparrow^Z] Z))$$ where the superscript is the amount of times the function is iterated, and the graph minor is the argument. $$a = \Psi_0(\chi_K(M))$$ where K represents the symbol for the first Weakly Compact Cardinal within Rathjen's countable ordinal notation system. $IGM^{var}(n) = GM(GM(\cdots GM(n)\cdots))$, which means the graph minor is iterated by the superscript's value. $$COWWD(n) = \\ (f_a(IGM^M(IGM^M(M [\uparrow^M] IGM^{M^Z}(M)))) + GM(GM(Z^{GM(M)}))) [\uparrow^{IGM^{M^Z}}] IGM^M(M) + Z [\uparrow^{IGM^{f_a(GM(M))}(f_a(M))}] GM(f_a(IGM^Z(f_a(IGM(M))))) + IGM^M(IGM^Z(Z^{IGM^{GM(M)}})) + f_a(E^M(IGM^M(M)))$$ The rules 1. The input, has to be a positive integer. 2. Any instance of $A [\uparrow^{var}] B$ means there are $var$ up arrows. $A [\uparrow^{var}] B$ is identical to $A \uparrow^{var} B$. 3. Any instance of $function^{var}(n)$ means that function is nested by the amount of the superscript and not exponentiated. When the nesting variable or value is 0, $function^{var}(n)$ equals the argument, and when it equals 1, it's not nested at all and only iterates once. 4. When evaluating the limit ordinal $\lambda$ within the FGH, it's resolved through ordinal notation fundamental sequences indexed by the current input argument: $f_\lambda(n)=f_{\lambda[n]}(n)$. But for collapsed large cardinal structures generated by Rathjen type multi-variable notations, the fundamental sequence is defined by standard proof-theoretic canonical assignments mapped down to the input. 5. The origin growth steps for successor ordinals are defined by $f_0(n)=n+1$ and $f_{\beta+1}(n) = f^n_\beta(n)$. 6. Vertical bars (like $|GraphMinor(n)|$) mean the maximum length the sequence can reach before the rules of the graph minor halts it. It's treated as absolute value when it's not a graph minor. 7. The order of operations for all equations or expressions is any math or expressions inside parenthesis or any container, then functional nesting or arguments, then graph minors, then Knuth arrows and any hyperoperations, then standard arithmetic (like addition or multiplication). 8. GM(n) is a function mapping $n$ to the maximum length of a subcubic graph minor sequence, with the exact same structural rules as the $SCG(n)$ function (to be specific, GM(n) is the maximum length of a subcubic graph minor sequence where the $i\text{-th}$ graph has a maximum of $n + i$ vertices). 9. Chained operations and hyperoperations are evaluated from right to left. Properties of COWWD(n) Similar to a lot of fast growing functions, $COWWD(n)$ has some properties that controls it's growth rate and bounds into the Extended Rathjen ordinal system. Property 1: COWWD is monotonic If $n$ is positive, the output will always increase if you increase the value of $n$. For example, $COWWD(n) < COWWD(n+1)$. This is because the function has Knuth arrows, M is monotonically dependent on Z, which is also monotonic, and the FGH is monotonic for increasing ordinal indexes, so the function is always monotonic. Property 2: Growth rate through the Extended Rathjen ordinal system COWWD eclipses functions bounded by the Extended Buchholz ordinals or lower ordinals. Why? Because the main thing COWWD has for growth is anchoring the bounding ordinal, $a$, to the countable limit of the Extended Rathjen notation system. The main COWWD equation also has 6 collapsing functions, even some used in graph minors. The symbol K serves as a structural bound inside the formal notation, and M acts as a modifier within the $\chi$ function, giving $a = \Psi_0(\chi_K(M))$. Because the FGH is evaluated through the computable fundamental sequences of the EROS, COWWD is computable while using the strength of the K ordinal.
